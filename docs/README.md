# 知識大亂鬥 - 貓狗大戰

課堂互動問答遊戲，結合知識測驗與即時對戰。

## 功能特色

- **Excel 題庫上傳**：支援 .xlsx 格式，含題目、四選項、正確答案
- **雙人對戰**：貓咪隊 vs 狗狗隊，答對題目獲得攻擊權
- **拋物線攻擊**：拖曳角色瞄準、調整力道發射飛行物
- **三區域傷害判定**：中心命中 35、內側 25、外側 10
- **道具系統**：加倍傷害、恢復血量、護盾、放大武器
- **超級賽亞人變身**：啟用加倍傷害時觸發變身特效
- **決鬥模式**：一擊必殺，含地面燃燒、煙霧、熱浪特效
- **音效系統**：戰鬥配樂、技能音效，可獨立開關
- **題目設定**：支援依檔案順序或隨機出題
- **選項隨機排列**：防止記憶選項位置

## 技術架構

- 純前端（HTML + CSS + JavaScript）
- Canvas 2D 遊戲渲染
- SheetJS 讀取 Excel 檔案
- 無需後端伺服器，直接開啟 index.html 即可使用

## 檔案結構

```
├── index.html          # 主頁面
├── style.css           # 樣式
├── main.js             # 遊戲邏輯
├── assets/
│   ├── cat/            # 貓咪素材（idle, attack, hurt, taunt, super）
│   ├── dog/            # 狗狗素材（idle, attack, hurt, taunt, super）
│   ├── items/          # 道具圖示
│   ├── portraits/      # 角色頭像
│   ├── effects/        # 特效圖片
│   ├── background.png  # 街頭風格背景
│   └── wall.png        # 磚牆
└── docs/
    ├── README.md       # 本文件
    └── CHANGELOG.md    # 變更記錄
```

## 使用方式

1. 開啟 `index.html`（或透過本地伺服器）
2. 上傳 Excel 題庫（格式：題目、選項A-D、正確解答）
3. 選擇題目順序、遊戲模式、音效設定
4. 選擇陣營，開始遊戲
