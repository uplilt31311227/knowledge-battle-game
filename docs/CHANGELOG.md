# 變更記錄 - 知識大亂鬥：貓狗大戰

## [Unreleased]

## [v1.6.0] - 2026-04-04

### 變更
- 移除 zoom out 相機系統（簡化遊戲畫面）
- 軌跡預測改回固定點數顯示
- 修正角色面向方向，依據實際位置動態翻轉（左側面右、右側面左）

## [v1.5.0] - 2026-04-04

### 新增
- 瞄準時預測軌跡延伸至拋物線頂點，搭配漸淡效果
- 軌跡超出畫面範圍時自動 zoom out，顯示完整預測弧線
- zoom out 以底部中央為錨點，平滑過渡動畫

### 修復
- 修復 zoom out 時背景未填滿整個可見區域的問題

## [v1.4.0] - 2026-03-29

### 新增
- 決鬥模式（一擊必殺）含地面燃燒、煙霧與熱浪特效
- 音效系統：戰鬥配樂、技能釋放音效、音效開關
- 超級賽亞人變身（雙倍攻擊時觸發）
- 美式漫畫風格遊戲素材、背景、角色頭像、技能圖示
- 快打旋風風格血量條與扣血特效
- 一般模式場景與角色特效（微風蕭瑟風格、落葉飄動）
- 選項隨機排列功能
- 攻擊後角色隨機移動位置
- 貓狗嘲諷動畫圖片
- 放大武器道具

### 變更
- 改為三區域傷害判定系統（中心 35 / 內側 25 / 外側 10）
- 上傳頁面改為美式漫畫風格
- 優化課程上傳頁面 UI 佈局
- 提升飛行物飛行速度 10%
- 放大角色 20%、放大飛行物 50%
- 更新飛行物圖片為魚骨頭與狗骨頭
- 更新街頭風格背景並調整角色位置

### 修復
- 修復重新開始遊戲時忽略題目順序設定的問題
- 修復題目隨機與決鬥模式無法正確執行
- 修正飛行物速度調整方式
- 改進 Excel 上傳錯誤處理
- 超級賽亞人圖像去背與攻擊後立即恢復狀態
- 恢復重力加速效果
