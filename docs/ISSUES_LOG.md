### nul 檔案殘留

- **日期**: 2026-04-10
- **狀態**: 🟢 已解決

**問題描述**：
專案根目錄存在一個 `nul` 檔案，為 Windows 系統誤建。

**解決方案**：
刪除 `nul` 檔案，已在 `.gitignore` 中忽略。

### restart() 缺少屬性導致血條動畫異常

- **日期**: 2026-04-04
- **狀態**: 🟢 已解決

**問題描述**：
重新開始遊戲時，`hpFlash` 和 `lastHp` 屬性未重置，導致血條動畫閃爍。

**解決方案**：
在 `restart()` 方法中加入 `hpFlash` 和 `lastHp` 的初始化。
